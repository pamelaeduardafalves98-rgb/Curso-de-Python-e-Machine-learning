# 📘 Introdução ao Python e Conceitos Iniciais

Este material apresenta os conceitos fundamentais da linguagem **Python**, desde sua introdução até os primeiros passos na programação.

---

## 🚀 Sobre o Python

Python é uma linguagem de programação:

- Interpretada  
- De alto nível  
- Multiplataforma  
- De propósito geral  

### 💡 Principais aplicações

- Desenvolvimento Web  
- Automação de tarefas  
- Ciência de Dados  
- Inteligência Artificial  
- Computação científica  

---

## 🏛️ Histórico

- Criada por Guido van Rossum  
- Início: final da década de 1980  
- Primeira versão: 1991  
- Python 3 lançado em 2008  
- Nome inspirado no grupo Monty Python  

---

## 🧠 Estrutura Básica

Python executa instruções de forma sequencial (de cima para baixo):

    print("Meu primeiro programa em Python")

---

## 💬 Comentários

### Uma linha

    # Comentário simples

### Múltiplas linhas

    """
    Comentário longo
    ou documentação
    """

---

## 📦 Variáveis

- Não precisam de tipo declarado  
- Tipagem dinâmica  
- Valores podem mudar  

    nome = "Ana"
    idade = 20
    altura = 1.65

---

## 🔢 Tipos de Dados

| Tipo  | Exemplo       |
|-------|--------------|
| int   | 10           |
| float | 19.90        |
| str   | "Texto"      |
| bool  | True / False |

---

## 🔍 Verificando Tipos

    print(type(idade))

## ⌨️ Entrada de Dados

    nome = input("Digite seu nome: ")

⚠️ `input()` sempre retorna string.

---

## 🔄 Conversão de Tipos

    idade = int(input("Digite sua idade: "))
    altura = float(input("Digite sua altura: "))

---

## ➗ Operadores Aritméticos

| Operador | Função            |
|----------|------------------|
| +        | Adição           |
| -        | Subtração        |
| *        | Multiplicação    |
| /        | Divisão          |
| //       | Divisão inteira  |
| %        | Resto            |
| **       | Potência         |

    a = 10
    b = 3

    print(a + b)
    print(a * b)
    print(a ** b)
